Semantic HTML
Markup Application
header defines a container for introductory content or a set of navigational links
main specifies the main content of a document
footer defines a footer for a document or section
nav defines a set of navigation links
aside defines some indirectly-related content aside from the content it is placed in (like a sidebar)
figure specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
ficcaption defines a caption for a figure element and can be placed as a first or last child
article defines independent, self-contained content
section defines a section of a document: a thematic grouping of content, typically with a heading
details defines additional details that the user can view or hide
mark defines marked/highlighted text
summary defines a visible heading for a details element
time defines a date/time
